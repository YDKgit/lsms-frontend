# LSMS Frontend

연구실 안전 관리 시스템 프론트엔드 (React + Vite)

## 사전 준비
- Node.js 18+
- 백엔드가 **http://localhost:8080** 에서 실행 중이어야 합니다.

백엔드 설정은 [lsms](https://github.com/YDKgit/lsms) 저장소 README를 참고하세요.

## 실행
```bash
npm install
npm run dev
```

브라우저: **http://localhost:5173**

## 로그인 (테스트 계정, 비밀번호 `1`)
| ID | 역할 |
|----|------|
| admin | 시스템 관리자 |
| labmanager | 연구실 책임자 |
| safety | 안전관리팀 |
| labsafety | 연구실 안전관리자 |
| edumgr | 교육 관리자 |
| researcher | 연구원 |
